# TableRush — Competitive Analysis
_2026-06-12_

## Comparison vs Top Competitors

### Overcooked (Ghost Town / Team17)

| Dimension | Overcooked | TableRush |
|-----------|-----------|----------|
| Platform | Console/PC | Mobile browser (free) |
| Session length | 5-20 minutes | 3 minutes |
| Players | 1-4 (co-op) | Solo only |
| Core loop | Multi-task cooking + delivery | Multi-task service loop |
| Difficulty | Escalating level-by-level | Session-based with rush events |
| Learning curve | Steep (many mechanics introduced) | Gentle (context-sensitive taps) |
| Emotional core | Chaotic fun with friends | Solo mastery performance |
| Price | $20-40 | Free |

**Why a player would choose TableRush over Overcooked:**
- Mobile browser = no download
- 3 minutes = fits a commute
- Solo = no coordination required
- Free

**Why Overcooked is better:**
- Local co-op is fundamentally more fun
- More variety (level design, obstacles)
- Higher production value

---

### Diner Dash (Glu Mobile)

| Dimension | Diner Dash | TableRush |
|-----------|-----------|----------|
| Platform | Mobile app | Mobile browser |
| Session length | 3-5 minutes | 3 minutes |
| Core loop | Seating + serving + color-matching | Seating + serving + combo |
| Monetization | IAP (wait timers, boosts) | None |
| Visual style | Cartoon, detailed characters | Minimal but warm |
| Progression | Linear levels | XP + unlocks per round |
| Friction | Paywall at key moments | Zero |

**Why a player would choose TableRush over Diner Dash:**
- No ads, no IAP, no energy timers
- Faster session loop
- More immediate satisfaction (payment burst, near-miss saves)
- Combo system adds strategic depth Diner Dash lacks

**Why Diner Dash is better:**
- More polished visual characters
- More content (100+ levels)
- Established brand recognition

## TableRush’s Unique Position

**What no competitor does exactly:**
1. **Free, browser-based, 3-minute restaurant game** with genuine depth (combo system, special customers, rush hour)
2. **Performance framing** — the fantasy of being the best waiter in the room, not just completing levels
3. **Zero friction** — no download, no account, no IAP, no ads, loads in 2 seconds

**The unique value proposition:**
> *The only restaurant management game that takes 3 minutes and is completely free with no strings attached.*

## Competitive Weaknesses

| Weakness | Impact | Fix |
|----------|--------|-----|
| No social sharing | Players can’t show off scores | Add share button on Game Over |
| No leaderboard | No competition against others | Optional future feature |
| One restaurant layout | Gets repetitive after 10+ plays | Post-v1: second layout |
| No character personality | Customers are archetypes, not characters | Post-v1: names + reactions |
| 4-bar music loop | Audible repetition in 30s | Post-v1: second theme |

## Competitive Strengths to Double Down On

1. **Instant play** — faster than any competitor to first game action
2. **3-minute sessions** — genuinely more mobile-appropriate than Diner Dash
3. **Combo system** — adds mastery ceiling that Diner Dash lacks
4. **Rush Hour** — better drama/event system than most competitors
5. **Near-miss saves** — unique emotional peak moment
6. **Session stories** — Game Over screen narrative summary is distinctive
