# TableRush

Fast-paced restaurant management game. Seat customers, take orders, deliver food, collect payment — keep the rush alive!

## Play
Live: https://mordechain.github.io/tablerush/

## Run Locally
```bash
npm install
npm run dev
```
Open http://localhost:3000

## Build
```bash
npm run build
```
Output in `dist/`.

## How to Play
- **Tap a table** to interact (context-sensitive)
- Seated customer → takes order popup
- Pick a menu item → player walks to kitchen and cooks
- When food is ready → tap table again to deliver
- After customer eats → tap to collect payment
- Dirty tables → tap to clean
- Keep your combo alive for score multipliers!
